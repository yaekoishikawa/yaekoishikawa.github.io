Web Designer Yaeko Ishikawa

千葉県市川市在住のWeb デザイナー、石川八重子です。オンラインスクールでWeb制作について学び、金融業界、アパレル業界から憧れていたWebデザイナーへと転職活動中です。これからさらにWeb制作の技術や知識を身につけて、Webサイト運用や問題解決のアドバイスもできるデザイナーになりたいと思っています。人がふっと見たくなる、また来たくなるような、魅力あふれたサイトを作りたいです。

Skills

Web

HTML/CSSでコーディングができます。JavaScript、PHP、Wordpress も今後勉強予定です。変化に応じて、新しい知識をつねに勉強して、対応します。

Illust

ニューヨークで勉強したので、デザインが得意です。学校在籍のときに自作のポップアート、油絵絵画販売をセントラルパークで開催の際、すべて完売。

English

学校や仕事はアメリカ、ニューヨークが長いので英語が得意です。Webデザインやファッション、最先端の情報なども海外の記事を読んで勉強しています。
